Creating a Site

To quickly create a site, you can run the generator with the -n (--new) key.

mukha-ssg -n

You will need to answer a few questions, and the site will be created. The site is created with a simple theme and minimal settings.

To quickly create a site, you can run the generator with the `-n` (`--new`) key.

```bash
mukha-ssg -n
```

You will need to answer a few questions, and the site will be created. The site is created with a simple theme and minimal settings.
