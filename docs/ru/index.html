ГСС «Муха»

Быстрый и простой генератор статических сайтов.

English

Простота
Один файл, можно хранить вместе с исходниками сайта, требует только Node.js, без внешних зависимостей.
Автономность
Сайт совместим с любым хостингом, а открывается даже без сервера. Встроенный поиск работает локально.
Работа с данными
Данные — полноправная часть контента; работа с данными и генерация страниц без написания кода
