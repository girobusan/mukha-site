Установка

Три способа установки

Хуки

Хуки позволяют запускать произвольные скрипты во время генерации сайта. Их можно писать на любом языке.

Режим предпросмотра

Режим предпросмотра позволяет в реальном времени отслеживать вносимые в сайт изменения. Удобно использовать при интеграции с gulp.

Сохранение данных для клиентских скриптов

При помощи Java Script API данные можно пробросить в клиентский скрипт. Они будут сохранены, а после могут быть асинхронно загружены по запросу. При этом табличные данные будут сохранены в компактном формате.

О проекте

Муха — еще один генератор статических сайтов со своими фишками.

Встроенный поиск

В Муху интегрована индексация и поиск с использованием lunr.   Поиск работает независимо от способа доставки контента.

Подключение данных

Данные являются полноправной частью контента Мухи. Их можно использовать как в шаблонах, так и в тексте и даже немножко обрабатывать.
