Чтобы быстро создать сайт, можно запустить генератор с ключом `-n` (`--new`)

```bash
mukha-ssg -n
```
Нужно будет ответить на несколько вопросов, и сайт будет создан. Сайт создается 
с простейшей темой и минимумом настроек.
